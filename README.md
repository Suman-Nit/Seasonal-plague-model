# Seasonal-plague-model (Matlab scripts)

#**Repository Files and Descriptions**

**/CI_out_prob_Fig7a**: Generates confidence interval plots for outbreak probability.

**/MTE_Fig12**: Estimates mean time to extinction (MTE) for different initial infected populations.

**/Out_prob_Fig6a**: Computes time-varying outbreak probability for different initial infection scenarios.

**/Seas_prob_Fig8f_9f**: Generate sensitivity plot of outbreak probability under varying seasonal parameter values.

**/PE_mod_better**: Performs Bayesian parameter estimation using MCMC/DRAM methods.

**/model_lambda_time_avg**: Function file for estimating R0 for non autonomous system.

**/R0**: Main script file for estimating R0.

**/model_lambda**:  Function file for plotting R0 for different set of parameters.

**/Par_eff_R0_Fig5f**: Main script file for plotting R0 for different values of parameters.

**/Sample_path_det_sol_Fig4b**: Generate stochastic sample paths along with deterministic solution for the model.

**/Spillover_prob_Fig10_Fig3**: Generate figure for spillover probability and time dependent parameters.

**/Sea_spill_Fig11_f**: Generate spillover probability sensitivity plots for differnt values of parameters.


# Requirements

This analysis was performed in MATLAB.
